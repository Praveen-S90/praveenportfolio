# PRAVEEN. - Data Scientist & AI Engineer Portfolio Website

A complete, modern, responsive personal portfolio website designed for **Praveen**, an aspiring Data Scientist & AI Engineer. Built with Vanilla HTML5, CSS3, and modern ES6 JavaScript.

## Color Palette

- **Soft Ivory Background (`#EFE9E0`)**
- **Tropical Teal (`#0F9E99`)**
- **White Cards (`#FFFFFF`)**

## Contact Form

The contact form uses mock success behavior only. It validates inputs, shows a loading state for 1.5 seconds, displays Praveen's success message, and resets the form without sending data to any backend.

## Local Execution

Open `index.html` in any browser, or use a local dev server such as Live Server to preview the portfolio.
